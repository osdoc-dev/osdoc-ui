## OSDOC-UI

基于 React 研发的一款适用于企业级的移动端 UI 基础组件与业务组件库。

## 特性

- 📦 开箱即用的高质量 React 组件。
- 🌈 形成规范和标准，降低设计、开发的人力成本和沟通成本。
- ⚙️ 配合工具体系，帮助业务快速完成高质量的产品原型。
- ✨ 使用 TypeScript 开发，支持静态检查，提升开发体验。
